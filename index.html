THE DATA VISUALIZATION

It is a period of data saturation. With a few clicks, we can be waist deep into any topic we desire. But the question remains...What do we do with all of that information?

Enter data visualization and the D3 JavaScript library.

The benefits of data visualization are manyfold. One important advantage of using D3 is the ability to interact with the data in meaningful ways.

Seeing the data layed out in different formats can help us interpret trends, and ensures faster comprehension of relationships.

All of the following data was retrieved from the Star Wars API (swapi.com). All charts were constructed with JavaScript, D3 and CSS. Please scroll down and enjoy the Star Wars Universe as you've never seen it before....
